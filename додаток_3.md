## 3. РЕАЛІЗАЦІЯ

### 3.1. Новий компонент SmartTable
**Файл:** `/home/franco/InTrack/frontend/src/components/SmartTable.jsx`

**Функціонал:**
- **Розтягування колонок:** користувач може тягнути межу колонки для зміни ширини
- **Збереження ширини:** ширина колонок зберігається в localStorage (`smarttable_widths_{table}`)
- **Обмеження розтягування:** перша колонка не розтягується вліво, остання з діями - вправо
- **Авто-ширина:** колонки автоматично підлаштовуються під контент
- **Обрізання тексту:** довгий текст обрізається з `...`, tooltip при наведенні
- **Пагінація:** опції 50/100/200/500/1000/10000
- **Клік на рядок:** перехід на деталізацію (через prop `onRow`)

**API:**
```jsx
<SmartTable
  columns={columns}           // Обов'язковий: масив колонок
  dataSource={data}          // Обов'язковий: дані
  rowKey="id"                // Обов'язковий: унікальний ключ
  loading={false}            // Опційно: індикатор завантаження
  storageKey="assets"        // Опційно: ключ для localStorage
  onRow={(record) => ({      // Опційно: клік на рядок
    onClick: () => navigate(`/assets/${record.id}`)
  })}
  pagination={{               // Опційно: кастомна пагінація
    current: 1,
    pageSize: 50,
    total: 1000
  }}
/>
```

**Властивості колонок:**
```jsx
{
  title: 'Назва колонки',
  dataIndex: 'field',
  key: 'unique_key',
  width: 150,              // Початкова ширина
  resizable: false,        // Заборонити розтягування (для останньої колонки)
  fixed: 'right',          // Зафіксувати справа (для кнопок дій)
  ellipsis: true,          // Обрізати текст з ... (за замовчуванням)
  sorter: true,            // Дозволити сортування
  render: (text, record) => {} // Кастомний рендер
}
```
